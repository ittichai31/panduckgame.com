# Cat Animations

Generated from `cut_sprites`. Each row folder became one looping animation.

Folder structure:

```text
animations/cat-name/sheet-1/cat-name_sheet-1_row-06_sitting.gif
animations/cat-name/sheet-1/cat-name_sheet-1_row-06_sitting.webp
```

Sheet 1 labels: row-00 idle-original, row-01 walking, row-02 running, row-03 jumping, row-04 falling, row-05 gliding, row-06 sitting, row-07 holding, row-08 petting, row-09 angry, row-10 climbing.

Sheet 2 labels: row-00 idle, row-01 sleep, row-02 grooming, row-03 lying-down.

Frame duration: 160 ms. Loop: forever.

Use GIF for easiest compatibility. Use WebP when available for cleaner transparency and smaller files.
